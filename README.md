# Учебная практика: Программирование на Python (ГУАП)

В данном репозитории представлены результаты выполнения цикла практических работ по дисциплине «Учебная практика». 
Цель работ — освоение инженерного подхода к разработке на Python: от простых вычислений до реализации численных методов.

## Описание работ

### Лабораторная работа №1: Основы вычислений
* **Задача:** Реализация программы для вычисления сложного арифметического выражения (Вариант №11).
* **Ключевые навыки:** Работа со стандартной библиотекой `math`, написание пользовательских функций (реализация котангенса).

### Лабораторная работа №2: Обработка данных и исключений
* **Задача:** Создание отказоустойчивой программы для вычисления значений функции при различных входных параметрах.
* **Технические особенности:**
    * Валидация пользовательского ввода через блок `try/except` (обработка `ValueError`).
    * Реализация механизма обработки неопределенностей (вывод `NaN` при невозможности вычисления).
    * Форматированный вывод результатов в виде структурированной таблицы.

### Лабораторная работа №3: Численные методы
* **Задача:** Вычисление значения функции sqrt{1-x} через разложение в бесконечный сходящийся ряд.
* **Алгоритмическая сложность:**
    * Использование **рекуррентного метода** вычисления членов ряда. Это позволило избежать операций с огромными числами (факториалами) и предотвратить переполнение памяти.
    * Достигнута точность вычислений **10⁻¹⁰**.
    * Оптимизация количества итераций цикла для достижения заданного порога точности.
